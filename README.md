# Project-PY1





















✊✋✌️ Option C - Tournoi Pierre-Papier-Ciseaux
Cas d'usage professionnel : Les algorithmes de tournoi (élimination directe, bracketing) sont utilisés dans les systèmes de compétition sportive, les plateformes de e-sport, et les systèmes de matching (Tinder, LinkedIn matching de profils). Ce projet enseigne la gestion d'algorithmes complexes.

Fonctionnalités principales :

Tournoi à 4, 8 ou 16 participants (humains + IA)
Élimination directe avec bracketing automatique
Match en 3 manches gagnantes
Affichage du bracket complet et podium final
Difficulté : ⭐⭐⭐ (Avancée - algorithmes et structures de données)
