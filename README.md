# Pierre 🗿 Feuille 🍃 Ciseaux ✂️


## -- Description --

Le projet est un tournoi de Pierre-Feuille-Ciseaux pouvant être être de 4, 8 ou 16 joueurs (humaine ou ia). Chaque match se joue en 3 manche gagnantes. À la fin du tournoi, un bracket ainsi qu'un podium sont affichés.


## -- Prérequis --

Pour ce qui est des prérequis, il est uniquement nécessaire d'avoir Python à jour.


## -- Répartition des rôles --

***Carlo*** : Structure du bracket à élimination directe et afficache du vainqueur. \
***Basile*** : Création des profiles des joueurs et logique des matchs.
